# Estándares de UI
- **Pagos:** El componente de cobro debe permitir entradas múltiples (split payments) sumando hasta el total de la orden.
- **Factura:** El diseño de la factura debe ser apto para impresoras térmicas (80mm) y formato PDF para WhatsApp.
- **Idioma:** Interfaz en español, código y variables en inglés (camelCase).
- **Diseño:** Usar Bootstrap 5  para estilos modernos y responsivos.
